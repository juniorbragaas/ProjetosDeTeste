TG-Envios

Tecnologias adotadas

  • ASP.NET Core e C# Plataforma e Back-End utilizado
  • Angular and TypeScript Código para intregação de eventos do lado do Cliente
  • HTML5 Front End da aplicação
  • Bootstrap Layout e Estilos
  • SQL Server Gerenciador de Banco de dados

Pendências

  • Identificação de Parâmetros
  • Acesso ao Web Services de processamento
  • Criação Serviço Importação dos Registro para o banco novo
  • JOB para processamentos do Registro e exibição de Log de Atividades

The ClientApp subdirectory is a standard Angular CLI application. If you open a command prompt in that directory, you can run any ng command (e.g., ng test), or use npm to install extra packages into it.
